## Installation

- Clone the repository
    ```bash
    git clone https://github.com/Codedwells/innoloft-dashboard.git
    ```
- Install dependencies
    - Using npm
        ```bash
        npm install
        ```
    - Using yarn
      ```bash
      yarn install
      ```
    - Using pnpm
        ```bash
        pnpm install
        ```

- Start the server
    ```bash
    npm start
    ```
- Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
